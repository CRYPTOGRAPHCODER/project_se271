작업 환경
open-source qt creator 4.4.0
Build : Desktop qt 5.9.3 MinGW 32bit


 * 이벤트 리스트

 * 형식
[확률] / [이벤트 이름] / [이벤트 메세지(길어도 됨, 줄 구분은 엔터가 아닌 \n으로)] 
  / 넣었으면 하는 효과 (아래 예시들은 코드로 작성되어 있지만, 코드로 모르면 말로 설명해도 가능)

* 수정 가능한 변수 및 주의점
플레이어 스탯
 체력 - 최대 체력 stats[0]
 회복 - 휴식시 회복량 stats[1]
 이학 - 수학, 과학 관련 과목 stats[2]
 공학 - 공학 관련 과목 stats[3]
 문학 - 문학 관련 과목 stats[4]
 사회 - 사회 관련 과목 stats[5]

라이프 - 1000 + 체력 스탯 * 100 life

장비 equipment[i]

돈 money

*** 난이도 - 게임의 전반적인 난이도. 1에서 시작해서 매 턴 1.009씩 곱해지면서 증가함 level

*** 랜덤 - rnd_d()는 0에서 1의 값을 반환


이벤트에서 선택지를 통해 다른 이벤트로 넘어가는 기능도 구현 가능함  

 > 친구 만나기

 > 교수님 만나기

 > 동아리방 가기

 > 그냥 돌아다니기

 > 일하기
100% / 노동의 기쁨 / 당신은 열심히 일했습니다.
  / life -= (int)(-this->level*200*(rnd_d()/2+0.75)), money += (int)(this->level*50000*(rnd_d()/2+0.75)) : 체력 소량 감소, 돈 다량 증가

 > 휴식
100% / 휴식 / 집에서 휴식합니다. 체력을 회복합니다.
  / life+=(int)(pl.get_stats()[1]*100*(rnd_d()/2+0.75))) : 체력 다량 회복

 > 공부하기
100% / 공부 / "당신은 열심히 공부했습니다.","방학이지만 집중해서 공부했습니다.","게임을 하고 싶은 열망을 누르고 공부했습니다.","오늘은 왠지 공부가 잘 되는군요.", "재밌게 공부했습니다."
  / life-=(int)(-this->level*150*(rnd_d()/2+0.75)), stats[(int)(rnd_d()*6)] +=(int)(rnd_d()*1.5+1) : 랜덤 스탯이 1 혹은 2 증가
