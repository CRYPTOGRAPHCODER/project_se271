 * Team Project for 2017_se271

 * Compile Environment
open-source qt creator 4.4.0
Build : Desktop qt 5.9.3 MinGW 32bit

 * What is this
수강신청 시뮬레이터 2017은 대학생이 되어 수강신청을 무사히 해 학기를 살아남아 졸업하는 게임입니다.
다양한 액션을 통해 과목에 대한 정보를 얻거나, 체력과 공부 스탯을 길러서 어려운 과목이 와도 버텨내 보세요!


 * Features
 - 100여 종류 이상의 다양한 이벤트
 - 현실적인 수강신청
 - 제한 없는 행동 가능
 - 개강 직후 종강이 찾아옴
 - 특별한 아이템들
 - 필요한 내용을 적을 수 있는 메모장


 * Tips
 - 오른쪽의 콘솔에서 원하는 내용을 드래그 한 뒤, 왼쪽 메모장으로 드래그 앤 드롭 하면 내용이 복사됩니다.
 - 체력을 기르면 어지간한 과목에도 문제 없을 지도 모릅니다.
